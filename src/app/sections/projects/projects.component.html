Menu Zone

Aplicación desarrollada con Flutter que permite a los usuarios encontrar el típico "menú del día" español a su alrededor.

Funcionalidades

  • Muestra restaurantes por distancia usando geohashes Usa la API de Google
  • Maps para cargar la información predeterminada de los restaurantes
  • Gestiona el estado global de la aplicación con "Provider"

IG4 Agricultura Razonada

Aplicación desarrollada para IG4 Agronomía usando Ionic. El objetivo de esta aplicación es mostrar a los agricultores información organizada en gráficos sobre las variables, como el clima, que afectan sus cultivos.

Funcionalidades

  • Usa gráficos (Highcharts) para gestionar los datos
  • Soporte multilenguaje con ngx-translate
  • Conexión con la API de la empresa (HttpClient)
